# Model Training Module

### Train the model

- `python3 main.py` can run the program
- Hyper parameters tuning can be adjusted in `main.py`
- A sh file can be used to submit the job to compute node on Gadi
- This python program needs training data from `../data_collect` folder
- This python program will output the model file and config file. Move them to `../library_frame/build/library` for the test program to use
